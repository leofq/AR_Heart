---- Intro ----

This application has been implemented using cognitivism learning theories.

The goal is to learn the anatomy of the human heart.

Use the marker to spawn in the 3D model and move around to learn.

---- Instructions ----
1. Navigate to scenes folder
2. Open ARHeart
3. Press play
4. You will need to use the marker (in assets) to spawn AR objects if using iOS. Windows does this by default.

Marker: MarkerIcons03
